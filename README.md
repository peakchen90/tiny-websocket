# tiny-websocket
基于 node 实现一个微小版 WebSocket

## 运行 demo
```bash
# 安装依赖
yarn install

# 编译
yarn build

# 运行demo
yarn demo

# 然后用浏览器打开 `demo/index.html` 文件 (HTTP协议方式)
```
